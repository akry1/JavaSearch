A card stack<br /><br /><br />The card layout displays only one item at a time and is only interesting with interactivity. The other items are stored in a stack and the displayed item is one of the items of the stack. The name of the card layout is a reference to a playing card deck where you can see the card at the top of the stack and you can put a card on the top. The difference in the card layout is that the items in the stack keeps their order. When you use this layout, you must use this method to add items to the container, i.e. the applet:<br /><br /><br />void add(String itemId, Component item)<br />Adds an item to the container and associate the item to the id.<br /><br /><br />The card layout has several methods to change the currently displayed item:<br /><br /><br />void first(Container container)<br />Display the first item of the stack.<br /><br /><br />void next(Container container)<br />Display the item of the stack that is located after the displayed item.<br /><br /><br />void previous(Container container)<br />Display the item of the stack that is located before the displayed item.<br /><br /><br />void last(Container container)<br />Display the last item of the stack.<br /><br /><br />void show(Container container, String itemId)<br />Display an item by its id.<br /><br /><br /><br /><br /><br />Code listing 9.5: CardLayoutApplet.java<br /><br /><br /> 1 import java.applet.Applet;<br /> 2 import java.awt.CardLayout;<br /> 3 import java.awt.Label;<br /> 4 <br /> 5 public class CardLayoutApplet extends Applet {<br /> 6 <br /> 7     static final String COMPONENT_POSITION_TOP = "TOP";<br /> 8     static final String COMPONENT_POSITION_MIDDLE = "MIDDLE";<br /> 9     static final String COMPONENT_POSITION_BOTTOM = "BOTTOM";<br />10 <br />11     Label topLabel = new Label("At the top");<br />12     Label middleLabel = new Label("In the middle");<br />13     Label bottomLabel = new Label("At the bottom");<br />14 <br />15     /**<br />16      * Init.<br />17      */<br />18     public void init() {<br />19         setLayout(new CardLayout());<br />20         add(COMPONENT_POSITION_TOP, topLabel);<br />21         add(COMPONENT_POSITION_MIDDLE, middleLabel);<br />22         add(COMPONENT_POSITION_BOTTOM, bottomLabel);<br />23         ((CardLayout)getLayout()).show(this, COMPONENT_POSITION_MIDDLE);<br />24     }<br />25 }
