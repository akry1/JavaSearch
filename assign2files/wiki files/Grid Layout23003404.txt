This layout arranges components in the form of the table (grid). The number of rows and columns in the grid is specified in the constructor. They have been ignored because there are not enough items to fill them and the number of rows is prior to the number of columns. This means that when you specify a number of rows that is not zero, the number of columns is simply ignored. You should specify zero rows in order that the number of columns is taken into account.<br />A grid layout creates cells with equal sizes. So it can be used not only to display items as a grid but also to display two items with the same width or height.
