The ArrayList class supports three constructors. The first constructor builds an empty array list.:<br /><br /><br />ArrayList( )<br /><br />The following constructor builds an array list that is initialized with the elements of the collection c.<br /><br /><br />ArrayList(Collection c)<br /><br />The following constructor builds an array list that has the specified initial capacity. The capacity is the size of the underlying array that is used to store the elements.<br />The capacity grows automatically as elements are added to an array list.<br /><br /><br /> ArrayList(int capacity)
