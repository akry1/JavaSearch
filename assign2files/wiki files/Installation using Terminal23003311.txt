Downloading and installing the Java platform on Linux machines (in particular Ubuntu Linux) is very easy and straight-forward. To use the terminal to download and install the Java platform, follow the instructions below.<br /><br /><br /><br /><br /><br />1.<br />Open the Terminal window.<br /><br /><br />2.<br />At the prompt, write the following (your package manager may be other than APT, so change the command accordingly):<br /><br /><br /><br />Retrieving the java packages<br /><br />$ sudo apt-get install openjdk-7-jdk openjdk-7-doc<br /><br /><br /><br /><br /><br /><br />3.<br />All Java softwares should be installed and instantly available now.
