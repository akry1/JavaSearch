Introduced in 1998 as a quick fix to the former versions, version 1.2 was the start of a new beginning for Java. The JDKs of version 1.2 and later versions are often called Java 2 as well. For example, the official name of JDK 1.4 is The Java(TM) 2 Platform, Standard Edition version 1.4.<br /><br />Major changes include:<br /><br />Rewrite the event handling (add Event Listeners)<br />Change Thread synchronizations<br />Introduction of the JIT-Just in time compilers
