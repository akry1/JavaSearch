The new features and upgrades included into Java changed the face of programming environment and gave a new definition to Object Oriented Programming (OOP in short). But unlike its predecessors, Java needed to be bundled with standard functionality and be independent of the host platform.<br />The primary goals in the creation of the Java language:<br /><br />It is simple.<br />It is object-oriented.<br />It is independent of the host platform.<br />It contains language facilities and libraries for networking.<br />It is designed to execute code from remote sources securely.<br /><br />The Java language introduces some new features that did not exist in other languages like C and C++.
