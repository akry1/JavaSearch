The Pattern class offers the function matches which returns true if an expression is found into a string.<br />For example, this script returns the unknown word preceding a known word:<br /><br /><br />import java.util.regex.Pattern;<br />public class Regex {<br />	public static void main(String[] args) {<br />		String s = "Test Java regex for Wikibooks.";<br />		System.out.println(Pattern.matches("[a-z]* Wikibooks",s));<br />    }<br />}<br />// Displays: "for Wikibooks"<br /><br /><br />The Matcher class allows to get all matches for a given expression, with different methods:<br /><br />find(): find the next result.<br />group(): displays the result.<br /><br />For example, this script displays the HTML b tags contents:<br /><br /><br />import java.util.regex.Pattern;<br />import java.util.regex.Matcher;<br /><br />public class Regex {<br />	public static void main(String[] args) {<br />		String s = "Test <i>Java</i> <b>regex</b> for <b>Wikibooks</b>.";<br />		Pattern p = Pattern.compile("<b>([^<]+)</b>");<br />		Matcher m = p.matcher(s);<br />		while(m.find()) {<br />			System.out.println(m.group());<br />			System.out.println(m.group(1));<br />		}<br />	}<br />}<br />/* Displays:<br /> <b>regex</b><br /> regex<br /> <b>Wikibooks</b><br /> Wikibooks<br />*/
