The JDK comes with a large number of tools related to the security features of Java. Usage of these tools first requires study of the particular security mechanisms. The tools are:<br /><br />keytool — To manage keys and certificates<br />jarsigner — To generate and verify digital signatures of JARs (Java ARchives)<br />policytool — To edit policy files<br />kinit — To obtain Kerberos v5 tickets<br />klist — To manage Kerberos credential cache and key table<br />ktab — To manage entries in a key table
