Sometimes it is useful to split a string into separate strings, based on a regular expressions. The String class has a split() method, since Java 1.4, that will return a String array:<br /><br /><br /><br />Code section 3.93: Order.<br /><br /><br />1 String person = "Brown, John:100 Yonge Street, Toronto:(416)777-9999";<br />2 ...<br />3 String[] personData = person.split(":");<br />4 ...<br />5 String name    = personData[0];<br />6 String address = personData[1];<br />7 String phone   = personData[2];<br /><br /><br /><br /><br />Another useful application could be to split the String text based on the new line character, so you could process the text line by line.
