In order for you to start using the JDK compiler utility with the Command Prompt, you would need to set the environment variables that points to the bin folder of your recently installed JDK. To set permanently your environment variables, follow the steps below.<br /><br /><br /><br /><br /><br />1.<br />To open System Properties dialog box use, the Control Panel or type the following command in the command window:<br /><br /><br /><br />System properties<br /><br />rundll32 shell32.dll,Control_RunDLL sysdm.cpl<br /><br /><br /><br /><br /><br /><br />2.<br />Navigate to the Advanced tab on the top, and select Environment Variables...<br /><br /><br />3.<br />Under System variables, select the variable named Path and click Edit...<br /><br /><br />4.<br />In the Edit System Variable dialog, go to the Variable value field. This field is a list of directory paths separated by semi-colons (;).<br /><br /><br />5.<br />To add a new path, append the location of your JDK bin folder separated by a semi-colon (;).<br /><br /><br />6.<br />Click OK on every opened dialog to save changes and get past to where you started.
