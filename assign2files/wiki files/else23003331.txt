else is a Java keyword. It is an optional part of a branching statement. It starts the 'false' statement block.<br />The general syntax of a if, using Extended Backus-Naur Form, is<br /><br />branching-statement ::= if condition-clause <br />                                    single-statement | block-statement<br />                       [ else <br />                                    single-statement | block-statement ]<br /> <br />condition-clause    ::= ( Boolean Expression )<br />single-statement    ::= Statement<br />block-statement     ::= { Statement [ Statement ] }<br /><br />For example:<br /><br /><br /><br /><br /><br /><br />if ( expression ) {<br />   System.out.println("'True' statement block");<br />} else {<br />   System.out.println("'False' statement block");<br />}<br /><br /><br /><br /><br /><br />See also:<br /><br />Java Programming/Keywords/if
