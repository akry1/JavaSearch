import is a Java keyword.<br />It declares a Java class to use in the code below the import statement. Once a Java class is declared, then the class name can be used in the code without specifying the package the class belongs to.<br />Use the '*' character to declare all the classes belonging to the package.<br />Syntax:<br /><br /><br /><br /><br /><br /><br />import package.JavaClass;<br />import package.*;<br /><br /><br /><br /><br /><br />The static import construct allows unqualified access to static members without inheriting from the type containing the static members:<br /><br />   import static java.lang.Math.PI;<br /><br />Once the static members have been imported, they may be used without qualification:<br /><br />   double r = cos(PI * theta);<br /><br />Caveat: use static import very sparingly to avoid polluting the program's namespace!<br />See also:<br /><br />Java Programming/Packages
