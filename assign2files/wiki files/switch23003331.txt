switch is a Java keyword.<br />It is a branching operation, based on a number.  	<br />   case 4:<br />      System.out.println( "Case: 4" );<br />      System.out.println( "Execute until break" );<br />      break;      <br />   default:<br />      System.out.println( "Case: default" );<br />      System.out.println( "Execute until break" );<br />      break;	      <br />}<br /><br /><br /><br /><br />The output from the above code is:<br /><br />Case: 3<br />Execute until break<br /><br />The same code can be written with if-else blocks":<br /><br /><br /><br /><br /><br /><br />int var = 3;<br />if ( var == 1 ) {<br />   System.out.println( "Case: 1" );<br />   System.out.println( "Execute until break" );<br />} else if ( var == 2 ) {<br />   System.out.println( "Case: 2" );<br />   System.out.println( "Execute until break" );<br />} else if ( var == 3 ) {<br />   System.out.println( "Case: 3" );<br />   System.out.println( "Execute until break" );<br />} else if ( var == 4 ) {<br />   System.out.println( "Case: 4" );<br />   System.out.println( "Execute until break" );<br />} else {<br />   // -- This is the default part -- <br />   System.out.println( "Case: default" );<br />   System.out.println( "Execute until break" );<br />}<br /><br /><br /><br /><br /><br />See also:<br /><br />Java Programming/Keywords/if
